# Dungeon-Build

Sobald ihr die ersten Dungeon-Sets farmen konntet, bietet sich folgender Build an: **Kraftvoller Ansturm** kombiniert mit **Magiekraftheilung** und dem Monster-Set **Ozezan das Inferno**.

Kraftvoller Ansturm ist zwar kein Dungeon-Set, kann aber in der Kaiserlichen Kanalisation gefarmt oder einfach bei einem Gildenhändler gekauft werden.

Die Sets können durch einen [Heilungsstab des Meisters](https://eso-hub.com/de/sets/grosse-verjungung){: target="_blank" rel="noopener" } ergänzt werden, oder Ozezan kann durch jedes andere verfügbare Heiler-Monster-Set ersetzt werden.

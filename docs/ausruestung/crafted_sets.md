# Hergestellte Ausrüstung

Hergestellte Sets könnt ihr selbst craften, sobald ihr die nötige Anzahl an Eigenschaften erforscht habt. Sie eignen sich daher besonders für den Einstieg oder als flexible Ergänzung zu Drop-Sets.

!!! note
    Set-Boni können sich mit Patches ändern. Genaue, aktuell gültige Werte prüft ihr am besten in ESO-Hub oder direkt im Spiel.

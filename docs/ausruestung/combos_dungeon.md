# Dungeon

## Magiekraftheilung + Kraftvoller Ansturm

**[Magiekraftheilung (SPC)](https://eso-hub.com/de/sets/magiekraftheilung){: target="_blank" rel="noopener" }** + **[Kraftvoller Ansturm (PA)](https://eso-hub.com/de/sets/kraftvoller-ansturm){: target="_blank" rel="noopener" }** + **[Heilungsstab des Meisters](https://eso-hub.com/de/sets/grosse-verjungung){: target="_blank" rel="noopener" }**, kombiniert mit **[Ozezan das Inferno](https://eso-hub.com/de/sets/ozezan-das-inferno){: target="_blank" rel="noopener" }** oder **[Sinfonie der Klingen](https://eso-hub.com/de/sets/sinfonie-der-klingen){: target="_blank" rel="noopener" }**

Diese Kombination kann immer und überall gespielt werden. Kraftvoller Ansturm wird nur auf der Backbar gespielt, wodurch auf der Frontbar Platz für den Heilungsstab des Meisters bleibt. Anstelle von Ozezan oder Sinfonie kann auch jedes andere Monster-Set gespielt werden.

## Magiekraftheilung + Meisterarchitekt

**[Magiekraftheilung (SPC)](https://eso-hub.com/de/sets/magiekraftheilung){: target="_blank" rel="noopener" }** + **[Meisterarchitekt (MA)](https://eso-hub.com/de/sets/meisterarchitekt){: target="_blank" rel="noopener" }** + **[Heilungsstab des Meisters](https://eso-hub.com/de/sets/grosse-verjungung){: target="_blank" rel="noopener" }**, kombiniert mit **[Ozezan das Inferno](https://eso-hub.com/de/sets/ozezan-das-inferno){: target="_blank" rel="noopener" }** oder **[Sinfonie der Klingen](https://eso-hub.com/de/sets/sinfonie-der-klingen){: target="_blank" rel="noopener" }**

Eine der stärksten Kombinationen für Dungeons, die aber genauso überall sonst gespielt werden kann. Meisterarchitekt wird nur auf der Backbar gespielt, wodurch auf der Frontbar Platz für den Heilungsstab des Meisters bleibt. Auch hier kann jedes andere Monster-Set gespielt werden.

# Raid

Normalerweise gibt es im Raid zwei Heiler. Beide Heiler sollten ihre Sets absprechen, um Dopplungen zu vermeiden. Für koordinierte Gruppen gibt es eine Vielzahl an möglichen Kombinationen. 

Hier die vermutlich häufigste Kombination für PUG-Raids.

### Heiler 1: Magiekraftheilung + Kraftvoller Ansturm

**[Magiekraftheilung (SPC)](https://eso-hub.com/de/sets/magiekraftheilung){: target="_blank" rel="noopener" }** + **[Kraftvoller Ansturm (PA)](https://eso-hub.com/de/sets/kraftvoller-ansturm){: target="_blank" rel="noopener" }** + **[Heilungsstab des Meisters](https://eso-hub.com/de/sets/grosse-verjungung){: target="_blank" rel="noopener" }**, kombiniert mit **[Ozezan das Inferno](https://eso-hub.com/de/sets/ozezan-das-inferno){: target="_blank" rel="noopener" }** oder **[Sinfonie der Klingen](https://eso-hub.com/de/sets/sinfonie-der-klingen){: target="_blank" rel="noopener" }**

Diese Kombination kann immer und überall gespielt werden. Kraftvoller Ansturm wird nur auf der Backbar gespielt, wodurch auf der Frontbar Platz für den Heilungsstab des Meisters bleibt. Anstelle von Ozezan oder Sinfonie kann auch jedes andere Monster-Set gespielt werden.

### Heiler 2: Tosender Opportunist + Jorvulds Führung (ROJO)

**[Tosender Opportunist (RO)](https://eso-hub.com/de/sets/tosender-opportunist){: target="_blank" rel="noopener" }** + **[Jorvulds Führung (JO)](https://eso-hub.com/de/sets/jorvulds-fuhrung){: target="_blank" rel="noopener" }** + **[Heilungsstab des Meisters](https://eso-hub.com/de/sets/grosse-verjungung){: target="_blank" rel="noopener" }**, kombiniert mit **[Erzdruide Devyric](https://eso-hub.com/de/sets/erzdruide-devyric){: target="_blank" rel="noopener" }**

ROJO ist eine beliebte Kombination für Gruppen, die ohne Slayer-Stacks spielen. Auf der Frontbar empfiehlt sich der Heilungsstab des Meisters, auf der Backbar ein Froststab aus dem Set des Tosenden Opportunisten – schwere Angriffe lassen sich damit deutlich einfacher setzen als mit einem Blitzstab. Erzdruide kann auch hier gegen jedes andere Monster-Set ausgetauscht werden.
